## Chapter 7｜檢疫逆算

從 C-9 升降艙出來的那一刻，祁渡有一種從冷死人的地獄，直接踏進了另一個高壓烤箱的錯覺。

第三十七基地的『主檢疫大廳』，在設計之初就被刻意打造成了一座猶如倒扣巨型漏斗般的白色劇場。

四面八方都是慘白到刺眼的無影燈。從盲區那種絕對黑暗的環境裡猛地出來，祁渡和另外幾名隊員幾乎被這光線刺得睜不開眼。眼前的景象更是帶著一種令人窒息的壓迫感——整個大廳被無數道高強度的透明防彈玻璃牆切割成了一個個獨立的隔離區塊。

每一條從深淵返回的回收隊列，都被無情地分流、驅趕進這些透明的籠子裡。

在這裡，沒有人情，只有流程。無數穿著全套防護服的檢疫人員穿梭其中，他們手裡的洗消噴槍噴灑出刺鼻的化學藥劑。人群的騷動聲、哀求聲、憤怒的叫罵聲，在穿過那一層層厚重的透明隔音牆後，全都被強行壓縮成了一種猶如幾千隻密封蜂箱同時發出的低頻嗡鳴，震得人腦仁發疼。

這是一個將人類物化、數據化到了極致的地方。只要系統判定你身上的污染指數超標，你就不再是一個人，而是一件隨時可以被就地銷毀的「危險污染源」。

因為是從最底層的 C-9 盲區回來，祁渡和第十二組的殘存隊員，被直接趕鴨子一樣趕進了條件最差的 C 區隔離室。

祁渡隔著玻璃牆往外看。就在他們對面的 A 區，那是軍方直屬精英部隊的優先洗消通道。那裡的設備全是最新的，檢疫床鋪是恆溫的，甚至連檢疫人員的態度都透著一絲恭敬，流程快得就像是在走過場。

而他們所在的 C 區，地上鋪滿了臨時墊起來的防水膠墊，角落裡赫然堆放著幾張生鏽的臨時拘束床，牆壁上掛滿了猶如蜘蛛網般雜亂的維生設備延長線。空氣中那股令人作嘔的消毒水味，濃烈得幾乎能嗆出人的眼淚。

沈洛已經在那裡等他們了。

女檢疫官的臉色從未如此難看過。她沒有理會那些正在對傷員進行緊急插管的醫療兵，而是直接走到祁渡面前，一把奪過了他手裡那支裝著骨白碎片的密封管，轉身就走向了隔離室邊緣那台看起來同樣有些年頭的重型『逆算分析台』。

逆算台，是基地裡唯一能對深淵未知物質進行初步成分拆解與溯源的可攜式主機。

沈洛將密封管粗暴地塞進了分析槽。

「嗡——」

機器的散熱風扇發出不堪重負的嘶吼，藍色的雷射掃描線開始一層一層地剝析那塊碎片。

不到十秒鐘，逆算台那塊佈滿輕微刮痕的螢幕上，伴隨著刺耳的提示音，瞬間亮起了三排猶如鮮血般刺 목적 紅色警告框：

**【判定結果：一、來源光譜未知（資料庫無匹配）。】**
**【判定結果：二、物理空間碼型重疊（帶有深淵底層特徵）。】**
**【判定結果：三、微觀波長指向——活體標記特徵。】**

「活體標記？」

祁渡就站在沈洛身後，他的心跳漏了一拍。他當然知道深淵裡的某些畸變怪物會在同類身上留下生物信號，但這塊硬邦邦的、像是從某個建築上敲下來的骨白碎片，怎麼可能會帶有活體的特徵？

「沈洛，這到底是什麼意思？」他忍不住開口追問。

沈洛死死盯著螢幕，沒有回頭，額角甚至滲出了細密的冷汗。她沒有回答祁渡的問題，而是十指如飛地在操作鍵盤上敲擊，試圖將這份初步的逆算資料，直接越級推送到官方最高級別的『聯合審查數據庫』去進行交叉比對。

「滴滴——退件。」

僅僅過了三秒，一個冰冷的紅色大叉就覆蓋了整個螢幕。

系統給出的退件理由簡單粗暴到令人髮指：
**【查無此權限。該資料列級已超過目前操作者權限上限。】**

「這不可能！」沈洛猛地一拳砸在操作檯的邊緣，清脆的金屬撞擊聲在隔離室裡迴盪。她轉過身，那雙總是冷靜理智的眼睛裡此刻卻燃燒著極度的憤怒，「我是 A 級檢疫官！平時就算是 S 級的污染樣本我也能查閱部分特徵！是誰？！是哪個混蛋突然把這塊碎片的檢疫權限往上鎖死了？！」

「不是誰鎖死了權限，沈洛。是這塊碎片本身，觸發了基地的一個『分流機制』。」

一個平靜、甚至帶著一絲悲憫的聲音，忽然從 C 區隔離室的另一道氣閘門方向傳來。

祁渡循聲望去。

不知何時，教團的祈函已經站在了那裡。她今天沒有穿那件標誌性的純白色祭祀袍，而是換上了一件深灰色的高領防護外披，整個人的氣質顯得異常肅穆，就像是剛剛結束了一場極其沉重、無法挽回的公開葬禮。

祈函緩步走到逆算台前，看了一眼螢幕上的退件訊息，語氣輕得像是一陣嘆息：

「從今天這個時刻開始，任何與『第零維修井』、以及『底層盲區特異物質』相關的檢疫流程，都不再歸你們常規的醫療部管轄。所有的權限和處置決定，已經全數移交給了剛剛成立的『深淵協議臨時委員會』。」

「哪個協議？」祁渡皺起眉頭，他有一種非常不祥的預感。三十七基地的官僚體系他這幾天已經領教夠了，每一個新名字的背後，往往都代表著一場重新分配生死的政治分贓。

祈函那雙宛如深潭般的眸子平靜地看著他，那種眼神裡沒有敵意，卻充滿了一種讓人毛骨悚然的宿命感。

「你不需要知道協議的具體內容。」祈函微微搖了搖頭，沒有正面回答這個問題，而是輕聲說出了一句更加致命的話，「你只需要知道……祁渡，你現在，已經在那份協議的『核心觀察名冊』裡了。」

就在這句話話音剛落的瞬間。

主檢疫大廳那巨大的環繞式廣播系統，突然爆發出一陣震耳欲聾的電子合成音。這不是普通的通知，而是帶有最高強制效力的全區廣播：

**【各區檢疫單位請注意。為維持基地整體穩定，防範底層污染擴散。請各區配合執行『高風險樣本指認程序』。】**

這個程序的名字聽起來很官方，但在場的每一個人都知道它意味著什麼。

這是一個殘酷到了極致的「棄車保帥」遊戲。

因為從盲區回來的人，身上都有暴露在未知污染中的風險。基地為了節省昂貴的深度檢疫資源，制定了這個不講理的規則——每一支被隔離的回收隊伍，必須從自己的隊員中，主動投票「指認」出一名最有可能成為「污染源」的候補者。

這名被指認的人，將會被當作最大的潛在威脅，直接被關進最底層的封閉銷毀室進行極端觀察，九死一生。

而作為「獎勵」，交出這名候補者的整支隊伍，其餘所有人都將獲得系統的「寬恕」，可以直接將整體的管制等級往下調降一級，換取優先離開這個見鬼的 C 區、獲得乾淨空氣和食物的權利。

「這不是在檢疫，這他媽是在逼我們殺人！」那名在底層跟祁渡一起對峙過軍方的黑市清道夫猛地站了起來，破口大罵。

沈洛沒有說話，她只是轉過頭，用一種極度複雜的眼神看著祁渡。她壓低了聲音，語氣中帶著一絲不易察覺的顫抖：

「祁渡，聽我的。照規定走。你們現在的處境太危險了，只要你們交出一個名字，你們就能先活著走出這個 C 區。離開這裡，我才能想辦法在委員會介入之前保住你。」

隨著沈洛的話音落下，隔離室裡原本就不安的空氣瞬間凝固了。

那兩名黑市清道夫，以及那名僅存的、沒有受傷的軍方回收手，幾乎是在同一時間，將目光齊刷刷地投向了角落裡那張臨時拘束床。

那名在 C-9 跌入裂縫的傷員，此刻正被綁在床上接受緊急輸液。他的臉色慘白得像個死人，嘴唇發紫，但他卻還保留著一絲清醒的意識。

在感受到隊友們那猶如實質般的目光後，傷員的身體開始劇烈地顫抖起來。他那隻沒有掛點滴的手，像個瘋子一樣死死地捂著自己胸前那塊還有些接觸不良的名牌螢幕，發出猶如野獸哀鳴般的聲音，彷彿那是他最後一根救命稻草，深怕自己的「名字」再一次被深淵剝奪。

這是一個多麼完美的「污染源候補」。

受傷最重、防護服破損、有過短暫的失名現象。只要祁渡作為這支隊伍的臨時隊長點一個頭，這場殘酷的遊戲就能立刻結束。所有的壓力、所有的嫌疑，都會瞬間從祁渡和其他人身上轉移，讓這個傷員去背負這個必死的黑鍋。

祁渡看著那張因為恐懼而扭曲的臉。

他的腦海裡閃過段九在配額大廳裡的嘲諷：下潛不是探索，是帶著名單去犧牲。

『只要點頭，我們就能出去。』祁渡在心裡對自己說。

然而，當他再次看見那名傷員絕望的眼神時，他發現自己根本無法輕易地點下那個頭。

他沒點。

在一片死寂中，祁渡轉過身，直視著隔離玻璃外那個正在拿著記錄板等待結果的軍方審核員。他的聲音不大，卻在空曠的隔離室裡顯得無比清晰而堅定：

「我們隊裡，沒有污染源。」祁渡一字一頓地說，「我不指認任何人。」

這句話一出，那名軍方審核員的眼角猛地抽搐了一下。他用一種看瘋子和死人的眼神看著祁渡，毫不猶豫地在手中的記錄板上按下了一個紅色的按鈕。

「十二組隊長拒絕配合指認程序。」審核員對著通訊麥克風冷冷地宣佈，「強制啟動懲罰機制。該隊伍全體成員，立刻升級為『第二級高危污染管制警戒』。」

「轟隆——」

隨著審核員的聲音落下，C 區隔離室原本就沉重無比的金屬電子閘門，伴隨著一聲刺耳的金屬摩擦聲，嚴絲合縫地徹底鎖死。

原本慘白刺眼的無影燈瞬間熄滅，取而代之的，是代表著深度隔離與隨時準備啟動銷毀程序的、猶如停屍間般陰森的冷藍色警示燈光。

那一刻，祁渡清楚地感覺到，自己後頸上那塊神經監測貼片，開始以一種前所未有的高頻率瘋狂發熱。強烈的刺痛感讓他幾乎要咬碎牙齒，彷彿有某種看不見的電流正在直接灼燒他的神經中樞。

同時，他手腕上的個人終端伴隨著劇烈的震動，彈出了一條全新的、帶著骷髏頭標誌的紅色狀態：

**【最高警告：個人深層身分待裁定異常。系統已將您標記為極度危險觀察對象。】**

沈洛重重地閉上了雙眼。她轉過身，背對著祁渡，雙手死死地撐在冰冷的逆算台上，肩膀因為極度的無力感而微微發抖。

她像是在拼命把某種即將爆發的怒氣和絕望硬生生地吞回肚子裡，語氣幾乎是從牙縫裡擠出來的：

「祁渡……你這個無可救藥的蠢貨。你知道你剛才那一句『不指認』，等於徹底放棄了什麼嗎？你放棄了你手裡最後一張可以用來討價還價的籌碼！」

「我知道。」

祁渡深吸了一口氣，強忍著後頸傳來的劇痛，他一步步走到逆算台前，看著那管被鎖死在機器裡的骨白碎片。他的眼神在冷藍色的燈光下顯出一種近乎偏執的冷硬：

「但我不想用別人的命，來換我自己的流程乾淨。如果這個基地的規矩就是要靠吃人才能活下去……那我就偏不守這個規矩。」

一直在一旁安靜觀察的祈函，此刻卻忽然動了。

她緩步走到祁渡和沈洛中間。她沒有去看祁渡終端上那刺目的警告，而是伸出那隻蒼白、沒有絲毫血色的手，直接越過了檢疫台的安全線，輕輕地按在了那支裝著碎片的密封管上。

「祈函，別碰！那上面可能有未知的輻射干擾！」沈洛大驚失色，剛想伸手去阻攔。

但已經來不及了。

就在祈函的手指隔著手套，觸碰到那層冰冷玻璃管壁的一瞬間。

原本因為「權限不足」而處於死機狀態的逆算台螢幕，突然發生了一陣劇烈的閃爍。那些代表著拒絕的紅色叉叉瞬間崩潰化為亂碼。緊接著，在沒有連接任何外部授權的情況下，螢幕的正中央，猶如幽靈般緩慢地浮現出了一行全新的、發著微弱綠光的加密訊息：

**【活體標記深度解析完成。】**
**【最終匹配結果：該物質深層信號源，高度疑似指向——目標人物：祁渡。】**

整個 C 區隔離室，在這一刻陷入了比 C-9 盲區還要恐怖的死寂。

祁渡整個人如遭雷擊，心口猛地一沉，彷彿有一隻無形的巨手死死攥住了他的心臟。

他的大腦一片空白。一台機器，一塊從深淵最底層的地板縫裡摳出來的、歷經不知道多少年歲月的骨白碎片，它的內部，竟然隱藏著一個指向「他自己」的生物標記？

祈函收回手。

她抬起頭，那雙深邃的眼睛靜靜地注視著祁渡。在那一刻，祁渡甚至從她的眼神裡看到了一絲連她自己都感到戰慄的恐懼。

祈函的聲音很低，低得只有他們三個人能聽見。但每一個字，都像是一把浸滿了冰水的刀子，狠狠地捅進了祁渡的靈魂深處：

「祁渡，你明白這意味著什麼嗎？」

祈函後退了半步，語氣中帶著一種令人絕望的確信：

「在底層盲區的那個十字路口，根本不是你去『找到』了那塊碎片。」

「是這塊碎片……從無盡的黑暗裡，『認出』了你。」
