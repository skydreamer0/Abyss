# S1V2 灰燼配給大綱（權力與生存擴張版）

> 目標：在 V1 確立生存邏輯後，V2 重點展示主角如何利用「資源稀缺」建立個人權威，並從「被動求生」轉向「主動收割」。

## 一、定位
- **Season**：S1 生存季。
- **Volume**：V2《灰燼配給》。
- **主角設定**：主角已轉變為「暴君型祁渡」，開始掌握配給權與武力威懾。
- **核心矛盾**：基地資源即將耗盡 vs 軍方的高壓配額 vs 黑市的生存槓桿。

## 二、敘事節奏（前中後三期）

### 前期｜權力卡位 (Chapter 1–5)
- **核心事件**：配給線斷裂引發暴動，主角被推上車隊隊長位置。
- **情節重點**：處決刺頭、突破無燈礦道、成功抵達中繼站。
- **成長表現**：建立「保護=佔有」的規則，初步覺醒深淵感知。

### 中期｜資源博弈 (Chapter 6–10)
- **核心事件**：車隊在荒野遭遇「徵收隊」攔截，黑市介入提供路權。
- **情節重點**：與段九的深度交易、活體礦壁的呼吸加劇、發現神屍神經叢。
- **成長表現**：從單純的駕駛者成長為「資源仲裁者」，決定誰能獲得「灰燼」。

### 後期｜異變昇華 (Chapter 11–16)
- **核心事件**：神屍偽復甦達到巔峰，中繼站面臨被消化危機。
- **情節重點**：QD-009 硬碟真相揭曉、主角第一次大規模「規則談判」、中繼站奪權。
- **成長表現**：身體異質化加重（符文融合），確立「我即深淵」的初級意識。

## 三、V2 章節功能規劃（16 章預測）

### Phase A 第一斷層：出發（1–4） [COMPLETED]
1. **配給崩潰**：平民區斷糧，基地進入緊急狀態。
2. **緊縮配額法**：軍方授權主角帶隊回收。
3. **車隊權標**：主角處決挑釁者，收服囚犯小組。
4. **無燈礦道**：環境初次異質化，拋棄電池誘捕盲喉獵鰭。

### Phase B 第二斷層：滲透（5–8）
5. **活體礦壁開始呼吸**：[DONE] 偽復甦爆發，強行衝入中繼站，胸口烙印符文。
6. **第一波徵收隊**：遭遇基地的競爭對手，雙方展開資源爭奪。
7. **黑市路權**：黑市勢力段九現身，提出以「禁技情報」換取物資。
8. **神經叢陷阱**：車隊偏離航線進入神屍核心器官。

### Phase C 第三斷層：質變（9–12）
9. **同步感染**：隊員開始出現大規模「同動症」，主角必須進行物理切除。
10. **硬碟解碼**：QD-009 內的女性身份確認（與主角過去記憶掛鉤）。
11. **中繼站淪陷**：環境徹底崩塌，中繼站指揮官叛逃，主角接管防務。
12. **禁術進化**：主角開發出能「固化局部空間」的常駐式能力。

### Phase D 第四斷層：收尾（13–16）
13. **回程封鎖**：基地拒絕車隊回港，判定其為「高危污染源」。
14. **瘋狂突擊**：主角帶領殘兵、掠奪黑市路權，殺回基地。
15. **配額重分**：主角掌握了基地的新命脈（神屍燃料）。
16. **暴君加冕**：主角成為基地的實際控制者之一，V2 完結。

    - **[卷級女性]**：引入 V2 核心女性角色（建議為中繼站的技術軍官或黑市派出的聯絡人）。
