# 深淵生態圈與微生物系統（初版）

## 一、設計原則
- 深淵生命不是「怪物名單」，而是能量流與污染流的動態網路。
- 每種生物都需定義：能量來源、繁殖路徑、天敵、可觀測徵兆、與人類風險。
- 微生物層為核心：多數大型異常其實是微生物群落失衡後的放大結果。

## 二、環境分層與生態區
### 1) 灰塵帶（基地周邊 0–5 km）
- 特徵：污染低至中，受人類活動干擾最大。
- 生態重點：拾荒性小型生物、寄生孢膜、基礎腐食菌。
- 風險：看似安全但最易出現慢性暴露與隱性感染。

### 2) 迴聲帶（5–30 km）
- 特徵：光照與聲波規律不穩，常有地形回寫。
- 生態重點：聲趨性獵食者、回聲共生藤、訊號擬態蟲群。
- 風險：認知誤導高，導航與隊伍通訊失真率上升。

### 3) 臟腑帶（30 km 以下）
- 特徵：高污染、高變形率，局部重力與時間漂移。
- 生態重點：巨型濾食體、神屍依附群落、相位漂移掠食者。
- 風險：高致死；任務多為一次性深入，不建議常規採集。

## 三、食物網與能量來源
- 初級來源 A：深淵噪訊沉積（類似輻射＋語義碎屑），由「噪食微生物」轉換成可用有機質。
- 初級來源 B：神屍滲出物（高密度蛋白與相位鹽），餵養特殊菌毯與巨型依附生物。
- 次級消費者：以菌毯、孢膜、腐食體為食的中型異生物（常形成遷徙波）。
- 頂級掠食者：不一定直接進食，常以擾亂訊號鏈與污染鏈維持棲地主權。

## 四、微生物系統（核心）
### A. 基礎類群
1. 噪食菌（Noise-feeders）
- 代謝：吸收規則噪訊，產生低熱高黏性生物膜。
- 生態角色：整個生態圈的「底層轉換器」。
- 人類影響：可用於低耗能濾材，但暴露過量會引發語義聽幻。

2. 鏡像菌（Mirror flora）
- 代謝：依附宿主神經訊號，複製其短期行為模式。
- 生態角色：促進群落同步，讓異生物短時間群體協調。
- 人類影響：感染者會出現「延遲模仿症」，行為慢半拍重演。

3. 鈍化菌（Blunt mold）
- 代謝：抑制高污染區域的突變速率。
- 生態角色：天然緩衝層，可暫時穩定營地外圍。
- 人類影響：可做臨時防疫塗層，但長期接觸會降低痛覺與情緒反應。

### B. 微生物交互
- 促進鏈：噪食菌增殖 → 提供鏡像菌基質 → 群落行為同步提升。
- 制動鏈：鈍化菌覆蓋率上升 → 突變下降 → 大型異常事件機率降低。
- 崩潰鏈：鏡像菌失控擴張 → 宿主辨識邊界崩潰 → 基地內部出現「同動症」。

### C. 傳播途徑
- 空氣：孢粉與冷凝霧。
- 接觸：義體接口、工具把手、防護服內層。
- 語義：被污染紀錄在朗讀、轉述後可成為低效率感染媒介。

## 五、中大型生物設計樣板
### 1) 鐵鬃馱獸（中型，群居）
- 食性：菌毯與腐食膜。
- 功能：可被半馴化為邊境運輸生物（高風險高報酬）。
- 威脅：受鏡像菌影響時會整群複製首領的自殺行為。

### 2) 盲喉獵鰭（中大型，伏擊）
- 食性：聲源導向捕食，偏好通訊器噪音。
- 功能：可反推安全頻段，作為反偵測研究對象。
- 威脅：會學習人類緊急口令並反向誘捕。

### 3) 神屍濾冠（巨型，定著）
- 食性：神屍滲出物與高污染懸質。
- 功能：是神屍碎片周邊的「生態工程師」，決定局部環境可居性。
- 威脅：成熟期會釋放大規模孢潮，重設整片棲地食物網。

## 六、人類聚落的生態工程
- 防疫四層：入口噴淋、裝備滅活、語音隔離、睡眠監測。
- 農業策略：以鈍化菌控制溫室污染，再以合成藻提供蛋白來源。
- 醫療策略：建立「菌相護照」，每次下潛前後比對個人菌群偏移。
- 黑市現實：回收商非法交易高活性菌株，常造成聚落級感染事故。

## 七、可直接用於劇情的事件鉤子
- 菌相赤字：基地全員菌相同時下跌，代表外圍鈍化菌被人為清空。
- 同動症夜班：守夜班成員開始同步重複同一句話與手勢。
- 孢潮封港：神屍濾冠進入成熟期，港區 72 小時全面停航。
- 假陰性事件：軍方新檢測劑失效，導致帶菌部隊進入市集區。
