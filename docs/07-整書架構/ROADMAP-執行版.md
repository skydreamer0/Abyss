## 2) 每卷 Phase 直接任務（可派工）

## V1《裂谷啟封》
- **Phase A 入局**：第三十七基地爆發把手事件，主角被臨時編入收容小隊。
- **Phase B 擴壓**：黑市、軍方、教團搶奪核心骨段，基地規則被逐條突破。
- **Phase C 崩裂**：收容程序遭內鬼破壞，主角被迫公開使用禁術。
- **Phase D 收束**：事件暫封，主角救城成功但失去至親記憶，簽下終身留用條款。

## V2《灰燼配給》
- **Phase A 入局**：邊境配給崩潰，平民區爆發氧化燃料暴動。
- **Phase B 擴壓**：主角護送車隊穿越活體礦道，連續遭武裝徵收隊攔截。
- **Phase C 崩裂**：聯盟內部有人出賣行車路線，車隊死傷慘重。
- **Phase D 收束**：基地聯盟成立，但糧藥供應被黑市壟斷。

## V3《白袍與黑市》
- **Phase A 入局**：三方啟動停火談判，實為地盤重分。
- **Phase B 擴壓**：教團以「聖蹟」收編民意，回收商壟斷污染素材。
- **Phase C 崩裂**：主角揭露聖蹟造假，談判桌爆發收容級災害。
- **Phase D 收束**：輿論戰勝利，但核心基地被合法獻祭。

## V4《逆鍊成條款》
- **Phase A 入局**：主角開始出現記憶回寫與身份錯位。
- **Phase B 擴壓**：軍方啟動「逆鍊成條款」人體試驗。
- **Phase C 崩裂**：主角確認魂譜原型來自神屍碎片。
- **Phase D 收束**：主角以自我逆鍊成翻盤，代價是人格邊界斷裂。

## V5《無名潮汐》
- **Phase A 入局**：多裂口勢力入侵，十二基地連鎖失壓。
- **Phase B 擴壓**：各基地同步熔斷，主角小隊分線救援。
- **Phase C 崩裂**：關門方案曝光：必須使用活體錨。
- **Phase D 收束**：聯軍暫穩戰線，主角接受成為候選活體錨。

## V6《門把協議》
- **Phase A 入局**：終局會議破裂，各勢力準備奪門。
- **Phase B 擴壓**：主角公開深淵真貌，世界秩序全面恐慌。
- **Phase C 崩裂**：門控系統失效，深淵回潮壓境。
- **Phase D 收束**：主角以身為門，簽下新秩序；世界存續但自由受限。

## 5) 主角 A 版（你指定：單主角／後宮向／無底線，後續固定）

### A 版：暴君型祁渡（正史）
- 角色定位：從災害處理者進化為權力收割者。
- V1 執行：靠救城建立威望，順手把軍方與黑市核心人員綁成私人圈。
- V2 執行：以配給危機作為籌碼，讓盟友必須「站隊 + 交人 + 交資源」。
- V3 前置：主動放出可控醜聞，清洗不服從派系，擴大個人法統。
- 關係策略：保護=佔有，忠誠不足就替換。
- 無底線行動清單：勒索、切割、假和解、公開處刑式談判都可使用。

### 後宮擴編規則（A版固定）
- 每卷至少新增 1 位核心關係角色（軍方/教團/黑市/外來者輪替）。
- 每卷至少有 1 次「關係反轉」：盟友變情人、情人變敵手、敵手再回收。
- 所有關係都必須影響權力分配，不做純裝飾角色。
