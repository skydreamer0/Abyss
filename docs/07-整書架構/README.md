# 07-整書架構

本目錄用於規劃小說從「整體走向」到「章節粒度」的敘事骨架。

## 內容索引
- [整本書設定與總綱（Season / Volume / Phase / Chapter）](./整本書設定與總綱.md)
- [ROADMAP 細部規劃（里程碑 / 交付物 / 完成條件）](./ROADMAP-細部規劃.md)
- [ROADMAP 執行版（六卷排程 / V1V2章節池）](./ROADMAP-執行版.md)

## 使用順序建議
1. 先確認「整本書設定」：主題、終局、長線懸念。
2. 再拆成 Season 與 Volume：安排每段的大衝突與情緒坡度。
3. 進一步細化 Phase：將每卷拆成可執行的劇情段。
4. 再進入 ROADMAP：把每階段交付物、完成條件、風險控管定義清楚。
5. 再用「ROADMAP 執行版」直接拉出每卷章節池與寫作順序。
6. 最後落到 Chapter：逐章確認目標、衝突、轉折與代價。
