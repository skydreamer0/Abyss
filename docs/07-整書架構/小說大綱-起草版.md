# 小說大綱起草版 v0.3

> 目的：依照既有 `WORKFLOW` 與 `ROADMAP`，提供「讀者可理解、主角不全知，且具備因果/升級/回應機制」的可開稿大綱。

## 0) 對齊結論（已消化既有流程）
- 採用 **Macro → Meso → Micro** 三層寫作。
- 章節固定使用 **G-O-C-R-S-H**（Goal/Obstacle/Cost/Reveal/Shift/Hook）。
- 節奏守則：每 2 章小反轉、每 4 章硬代價。
- 深淵資訊按層揭露：現象 → 流程 → 勢力 → 系統 → 真相。
- 禁則：不做無成本勝利，不連續兩章純資訊章。

## 0.1) 認知限制敘事規則（本版新增）
- **主角只知道眼前可驗證事實**：主角不負責解釋世界觀，只負責活下來。
- **讀者理解來自「他人說法 + 現場後果」**：同一事件至少給兩種互相衝突解讀。
- **每章最多 1 個關鍵解釋段**：避免上帝視角說明文。
- **關鍵資訊必附偏差來源**：誰說的、立場是什麼、可能騙人的點是什麼。

## 0.2) 結構邏輯工具（整合版）

### A. 核心五條（每章必檢）
1. **因果鏈推進（Cause → Effect）**：事件必須由前因觸發，不可憑空發生。
2. **壓力升級（Escalation）**：每次衝突都比前一次更嚴重，不做同級重複。
3. **Try–Fail 循環**：主角每次嘗試都帶出新問題或更高代價。
4. **世界回應（World Reaction）**：主角行動後，制度/勢力/環境必須反應。
5. **能力代價平衡（Power Balance）**：能力越強，限制越硬；禁止免費外掛。

### B. 強化五條（卷級強化）
6. **資訊節奏（Mystery Economy）**：現象 → 異常 → 規則線索 → 真相片段 → 全貌。
7. **場景功能化（Scene Function）**：每場景至少完成「推劇情/揭資訊/建角色/升壓力/埋伏筆」之一。
8. **伏筆回收（Setup → Payoff）**：前 1/3 放線索，中段錯解，後段反轉回收。
9. **視角落差（Dramatic Irony）**：讀者可短暫比主角多知道半步，製造懸念。
10. **三層衝突並行**：外部衝突 + 內部衝突 + 系統衝突同場推進。

- **V5《無名潮汐》**：多基地戰線同時崩潰，主角必須建立新聯盟與新規則。
- **V6《門把協議》**：主角以自身成為門的代價改寫世界秩序，換來存續但失去完整自由。

## 2) S1V1《裂谷啟封》卷級骨架（Macro）
- **卷目標**：讓新讀者在 8 章內理解深淵危險、流程與代價。
- **卷衝突**：基地要穩定封鎖裂谷，卻同時遭遇污染升級與勢力內鬥。
- **卷硬代價**：主角永久失去一段至親記憶。
- **卷末鉤子**：更高層的「門把協議」派系將主角列為核心樣本。

## 2.1) 角色功能卡（用來做「慢慢摸」）
- **主角（穿越者）**：只懂求生直覺，不懂術語；每次理解都靠代價換來。
- **檢疫官・沈洛**：提供流程層知識（偏保守、重紀律），資訊可靠但不完整。
- **教團見證官・祈函**：提供真相層暗示（偏神學、重隱喻），資訊有誇飾成分。
- **回收商・段九**：提供黑市實務（偏利益、重結果），資訊常夾帶私貨。
- **基地主控 AI「鹽井」**：提供可驗證紀錄，但在污染期會被改寫。

> 寫作策略：同一問題先給「沈洛版本」，再給「段九版本」，最後讓主角用行動驗證誰比較接近真相。

## 3) S1V1 四段推進（Meso）

### Phase A 入局（Ch1–4）
- 目標：建立危險感與三方勢力入場。
- 阻礙：資訊污染導致命令與現場不一致。
- 關鍵決策：主角選擇留在基地核心區，不隨撤離隊離開。
- 代價：主角初次出現記憶缺漏徵兆。
- 認知進展：讀者知道「污染會改寫資訊」；主角只知道「照流程也可能死」。

### Phase B 擴壓（Ch5–8）
- 目標：展示下潛→回收→檢疫→熔斷流程。
- 阻礙：流程被政治、宗教與黑市利益切割。
- 關鍵決策：主角接管熔斷節點，放棄個人撤離名額。
- 代價：同伴失名，且無法完全恢復。
- 認知進展：讀者理解流程全鏈；主角只掌握「哪一步最容易死人」。

### Phase C 崩裂（Ch9–12）
- 目標：把危機升到基地級，迫使主角越線。
- 阻礙：軍令與求生需求衝突，外來資本介入抬價。
- 關鍵決策：主角違令使用禁術。
- 代價：主角神經污染級別上升，失去部分身份錨點。
- 認知進展：讀者理解禁術代價機制；主角只知道「再用一次會忘掉某個人」。

### Phase D 收束（Ch13–16）
- 目標：完成短期封鎖，保住基地主體。
- 阻礙：檢疫餘震與內部究責。
- 關鍵決策：主角簽下終身留用條款以交換倖存者保全。
- 代價：主角永久失去至親記憶（不可逆）。
- 認知進展：讀者看見制度如何吞噬英雄；主角只剩「活著就得繼續服役」。

## 3.1) 壓力升級曲線（S1V1 共用）
1. 發現異常（黑雪、錯名、監控不一致）
2. 遭遇威脅（失名者失控、流程互斥）
3. 無法撤退（封鎖升級、名單改寫）
4. 損失資源（名額、清白覆核、隊友完整人格）
5. 生存危機（熔斷倒數、基地內戰）
6. 基地級危機（門把協議介入、系統層獵殺）

> 規則：每跨一級，至少增加一項「不可逆代價」。

## 3.2) 因果鏈主幹（Ch1–8）
- 發現監控異常（Ch1）→ 主角被留置（Ch1）→ 必須接受記憶覆核（Ch2）
- 記憶覆核留下缺口（Ch2）→ 失名者被錯誤處置（Ch3）→ 觸發把手近啟（Ch3）
- 反向座標指向禁區（Ch3）→ 三方簽封拖慢查核（Ch4）→ 禁區提前開啟事實曝光（Ch4）
- 為查真相主角下潛（Ch5）→ 盲區回收失控（Ch6）→ 神屍碎片進入檢疫鏈（Ch7）
- 檢疫鏈被政治化（Ch7）→ 熔斷被迫提前（Ch8）→ 主角以記憶支付啟動成本（Ch8）

## 4) 起稿章節池（Micro，補深度到前 8 章）

## Chapter 1｜裂谷醒轉
- **Goal**：搞懂自己身處何地並活過第一次封鎖。
- **Obstacle**：黑雪事件讓基地所有通道失序封閉。
- **Cost**：若硬闖將被判污染嫌疑，直接隔離。
- **Reveal**：深淵會改寫記錄與命名，資訊本身不可靠。
- **Shift**：原定撤離名單被改寫，主角變成留置人員。
- **Hook**：主角在監控回放看見「自己沒做過的行為」。
- **讀者理解設計**：由沈洛講「封鎖規程」；段九當場反駁「規程只保上層」。
- **Try–Fail / 世界回應**：主角嘗試按規程申訴失敗 → 被系統標註「高風險留置」。

## Chapter 2｜黑雪名冊
- **Goal**：證明自己沒有污染並取得臨時通行。
- **Obstacle**：檢疫官與教團見證規則互相衝突。
- **Cost**：主角需交出一段個人記憶做比對樣本。
- **Reveal**：污染可透過語詞傳播，錯誤命名會擴大災情。
- **Shift**：主角洗清嫌疑，卻被標記為高觀察對象。
- **Hook**：第一名「失名者」在檢疫區失控。
- **讀者理解設計**：祈函把失名者稱為「被召回」；沈洛堅稱是腦損傷，形成認知拉扯。
- **Try–Fail / 世界回應**：主角嘗試證明自己正常 → 通過檢疫但失去記憶樣本，後續被更嚴密監控。

## Chapter 3｜把手事件
- **Goal**：阻止失名者觸發裂谷二次開啟。
- **Obstacle**：軍方要就地處決，教團要活體收容，黑市要偷運樣本。
- **Cost**：主角若介入，將失去中立身份與保護。
- **Reveal**：裂谷核心的「把手」可被人為誤觸並放大。
- **Shift**：主角成功阻止觸發，但失名者留下反向座標。
- **Hook**：座標指向基地禁區與主角穿越落點重疊。
- **讀者理解設計**：不解釋把手原理，只展示「觸發後 90 秒內環境如何異常化」。
- **Try–Fail / 世界回應**：主角救下失名者避免處決 → 軍方啟動戰時條款，主角失去中立保護。

## Chapter 4｜三方簽封
- **Goal**：進入禁區查核座標真偽。
- **Obstacle**：三方勢力要求共同監督，行動效率被拖垮。
- **Cost**：主角必須簽下臨時責任條款，失敗就背鍋。
- **Reveal**：基地封鎖流程存在可被高層濫用的後門。
- **Shift**：查核成功，卻證實禁區曾被提前開啟。
- **Hook**：主角在禁區牆面發現「自己的舊簽名」。
- **讀者理解設計**：主角不懂舊簽名意義；由鹽井提示時間戳異常，留到後文回收。
- **Try–Fail / 世界回應**：主角要求單線查核被否決 → 三方改採聯審，行動時限縮短。

## Chapter 5｜下潛配額
- **Goal**：拿到下潛隊臨時名額，查明禁區提前開啟原因。
- **Obstacle**：配額被軍方與黑市內定瓜分。
- **Cost**：主角用掉唯一一次「清白覆核」資格。
- **Reveal**：下潛並非探索，而是「帶著可犧牲名單」回收。
- **Shift**：主角拿到名額，但被分配到死亡率最高路線。
- **Hook**：段九私下塞給主角一張過期地圖，疑似能避開監控。
- **能力代價**：主角獲得「路線判讀優勢」，代價是清白覆核機會歸零。

## Chapter 6｜回收盲區
- **Goal**：完成第一次回收並全員返回。
- **Obstacle**：舊地圖與現場拓撲不一致，隊伍陷入盲區。
- **Cost**：若強行回收，必須放棄一名傷員。
- **Reveal**：深淵地形會因觀測方式改變；「看見」本身是風險。
- **Shift**：主角用錯誤指令反而打開生路，但隊友開始遺失名字。
- **Hook**：回收箱裡多了一件不在任務清單上的神屍碎片。
- **世界回應**：基地立刻封鎖返回航道，回收隊全員改列臨時疑似污染者。

## Chapter 7｜檢疫逆算
- **Goal**：找出碎片來源，避免全基地被列為污染樣本。
- **Obstacle**：檢疫流程被高層插隊，資料遭到刪改。
- **Cost**：主角必須指認一名同伴為污染源候補。
- **Reveal**：檢疫不是純醫療，而是政治分流工具。
- **Shift**：主角拒絕指認，導致自己被升級為二級管制對象。
- **Hook**：祈函透露：碎片可能認得主角，而不是主角找到碎片。
- **伏筆回收**：Ch1 的監控異常與 Ch4 舊簽名被串連，暗示主角早被標記。

## Chapter 8｜熔斷值夜
- **Goal**：在熔斷程序崩潰前完成全線切割。
- **Obstacle**：軍令要求保設備，現場則要先保人命。
- **Cost**：熔斷需要主角的記憶片段作為啟動鑰。
- **Reveal**：禁術其實是「把個人記憶當燃料」的合法外掛。
- **Shift**：主角成功熔斷，基地暫時存活，但忘記了穿越前最重要的一個名字。
- **Hook**：鹽井在熔斷紀錄中標註：「該名字與門把協議關聯度 87%。」
- **能力代價 + 三層衝突**：外部（熔斷倒數）/內部（記憶流失恐懼）/系統（軍令保設備）同時壓上。

## 5) 寫作護欄（避免主角全知）
- 每章最多 1 次「主角推理成功」；至少 1 次「主角判斷失誤」。
- 專有名詞出現後，需在 2 章內用場景驗證，不准只靠解釋成立。
- 不允許主角直接講出世界真相；真相只能由衝突後果慢慢浮出。
- 每次揭露都要標記來源（軍方/教團/黑市/AI），讓讀者知道資訊帶偏見。


## 5.1) 場景功能檢核卡（寫完一場就勾）
- [ ] 推進劇情（局勢有變）
- [ ] 揭露資訊（新增可驗證線索）
- [ ] 建立角色（關係或信念改變）
- [ ] 提升壓力（時間、資源、敵意任一升級）
- [ ] 埋伏筆（可在 3–8 章內回收）

> 規則：若場景 0 勾，刪；僅 1 勾，考慮合併。

## 5.2) 單章運算模板（連載防發散）
- 因：上一章哪個後果逼出本章目標？
- 果：本章結尾改變了哪個系統狀態？
- 升級：壓力從第幾級升到第幾級？
- 失敗：主角本章的錯誤判斷是什麼？
- 回應：世界如何懲罰/利用主角行動？
- 回收：本章埋了什麼，預計哪章回收？
