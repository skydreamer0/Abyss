# 03-系統

本目錄管理所有「可操作機制」：裂印工學、接口義體、禁忌科技、污染分期、收容程序與人口補充工法。

原則：
- 能力必須量化。
- 代價必須可追蹤。
- 失控必須可敘事。
- 術語必須維持本作獨立系統，不借用他作框架。
- 所有高階技術需對應污染階段與處置流程。
