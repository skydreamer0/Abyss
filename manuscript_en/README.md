## 轉譯守則備註 (Adaptation Rules)
所有此目錄下的產出，均嚴格遵守 `.agent/skills/western_novel_adapter/SKILL.md` 的定義：
1. **Explicit Monologue**: 強烈且直接的內心吐槽/獨白 (Italics)。
2. **Pulp Prose**: 俐落的動作場景，拒絕過度華麗的形容詞。
3. **The Hook**: 重視每章結尾的懸念設計。
