# 生物多樣性名錄 v1（棲位版）

> 目的：把深淵生物從「零散怪物點子」升級為可運行生態網，確保每個棲位至少有候選物種。

## 0) 基本資訊規格（所有物種至少要有）
- 學名代號：由觀測站統一編碼（例：AF-MI-013）。
- 污染等級：I–V（I 最低、V 最高）。
- 棲地層級：灰塵帶 / 迴聲帶 / 臟腑帶 / 跨層遷徙。
- 食性與天敵：至少寫出一條能量來源與一個天然制衡者。
- 繁殖條件：季節窗、污染閾值、溫濕度或聲場需求。
- 對人類關係：可利用價值 + 主要副作用。

## 6) 基本生命史補充（新）
- 繁殖季節窗：
  - 灰塵帶：以乾季末為主（資源集中，競爭高）。
  - 迴聲帶：以低噪潮週期為主（訊號干擾下降）。
  - 臟腑帶：無固定季節，受投放事件觸發。
- 幼體高死亡群：盲喉獵鰭、黑冠折骨鳥、盲井脈龍（幼體期死亡率 > 70%）。
- 爆發型群落：鏡像菌 + 灰脊寄背蟲共同升高時，最易出現聚落級行為異常。

## 7) 棲位空缺（後續補完優先）
- 可跨層遷徙且可追蹤的「標記物種」仍不足。
- 深層淡水生態資料仍少，補給規劃風險高。
- 巨型掠食者求偶期行為資料不足，難建立年度預警模型。

## 8) 生態平衡指標（監測用）
- 噪食菌 / 鈍化菌 比值（N/B Ratio）：> 3.0 代表失穩風險上升。
- 中型草食者遷徙密度：低於季節均值 40% 代表上層資源崩落。
- 巨型定著體孢釋放次數：月內 > 2 次即啟動港區封鎖預案。
- 夜行授粉者密度：低於基線 50% 代表翌月植被授粉失敗風險高。

- 高空夜行授粉者數量不足（目前僅鉤足燈蛾主導）。

## 8) 創作規則（避免設定崩壞）
- 新增物種必填：食性、天敵、繁殖條件、棲地、污染風險。
- 不允許「全能生物」：每種生物至少有一個明確脆弱點。
- 任何生物價值（可食/可藥/可武）都必須綁定副作用。
